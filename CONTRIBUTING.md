# Contributing Guide

欢迎参与改进本项目。

## What you can contribute

- Add new parking data examples
- Improve UI design
- Add map integration
- Add online booking workflow
- Improve documentation
- Fix bugs

## How to contribute

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Submit a pull request.

## Data format

Parking data is stored in `data/parking.json`. Please keep the JSON format valid.
